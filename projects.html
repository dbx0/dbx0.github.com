Olleak

olleak discovers publicly exposed Ollama instances via Shodan, probes every available model with a chat message, and records the results, including response language, token counts, and timing in a PostgreSQL or SQLite database.

LODC - Living Off Default Credentials

The Living Off Default Credentials (LODC) project is a community-driven collection of default credentials used by various vendors and products. This resource is designed for security researchers, penetration testers, and system administrators to understand and mitigate the risks associated with default credentials.
